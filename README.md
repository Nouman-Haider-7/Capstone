# Capstone
Capstone project for FlyRank AI Engineering internship
## What is this?
StageWay is a casting and talent marketplace that connects directors with actors, built as a final year capstone project using the MERN stack.

## Getting Started
This project is currently in early development. Setup instructions will be added as the frontend and backend are built out.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
